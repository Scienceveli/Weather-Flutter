## Run Locally

Clone The Repo Using 

```bash
  git clone https://github.com/Scienceveli/Brand-O-Flutter
```

Go To Main Clone Directory

```bash
  cd Flutter-Projects
```

Go to Project directory

```bash
  cd project_name
````
Install dependencies

```bash
  flutter pub get
```

Start or Run 

```bash
  flutter run
```
